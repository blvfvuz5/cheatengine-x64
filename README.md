# cheatengine-x64

Modern, minimal GitHub Pages download guide for the Windows x64 release flow.

## Overview

This project is a lightweight static website built with plain HTML, CSS, and JavaScript. It provides a clean landing page that guides visitors to the GitHub Releases page for the latest Windows x64 executable.

## Features

- Modern dark tech-style landing page
- Minimal glass-like card UI
- Responsive layout for desktop and mobile
- Subtle scroll reveal animation
- Soft cursor glow interaction
- Simple GitHub Releases guide structure

## Download Guide

1. Open the GitHub repository
2. Go to the Releases section
3. Select the latest version
4. Download the `.exe` file
